Privacy Policy

Last updated: October 20, 2025

Overview

Keep Alive ("the App") is designed to monitor and restart selected applications on your Android device. This privacy policy describes how we handle information in connection with the App.

Information Collection and Use

The App operates entirely on your device and does not collect, transmit, or store any personal information to external servers by default.

Local Data

The App stores the following information locally on your device:

  • List of applications you've selected to monitor
  • App settings and preferences (check intervals, etc.)
  • Local logs of app monitoring activity

Optional Features

The App includes optional features for remote logging and heartbeat monitoring. These features are:

  • Disabled by default
  • Require explicit user configuration
  • Only transmit data when you provide API endpoints or configuration

If you choose to enable these features, you are responsible for any data transmitted to third-party services you configure.

Permissions

The App requires various Android permissions to function. These permissions are used exclusively for the App's core functionality of monitoring and restarting applications. For a detailed list of permissions and their purposes, please see the main app page.

Third-Party Services

The App does not integrate with any third-party analytics or advertising services. Any third-party services are only used if you explicitly configure optional features (remote logging, heartbeat monitoring).

Data Security

All app data is stored locally on your device using Android's standard security mechanisms. We do not have access to your device or any data stored by the App.

Changes to This Privacy Policy

We may update this privacy policy from time to time. We will notify you of any changes by posting the new privacy policy on this page and updating the "Last updated" date.
